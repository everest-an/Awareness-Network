# Filecoin UI 设计细节分析

## 核心设计特点

### 1. 顶部导航栏
- **固定定位**: 始终固定在顶部
- **透明背景**: 使用半透明背景 + backdrop-blur
- **Logo左侧**: Filecoin logo在左侧
- **导航链接居中/右侧**: Store Your Data, Provide Storage, Build on Filecoin, Blog
- **简洁**: 没有过多装饰，干净利落

### 2. Hero区域
- **巨大的3D地球**: 占据视觉中心，带有发光效果和网格线
- **标题字体**: 相对较小但清晰，不是超大字体
- **副标题**: 简洁的一句话描述
- **CTA按钮**: 简单的文字链接，不是大按钮

### 3. 字体比例
- **标题**: 约36-48px，不是超大字体
- **副标题**: 约18-20px
- **正文**: 约14-16px
- **字重**: 标题500-600，正文400
- **行高**: 1.5-1.6

### 4. 间距
- **大量留白**: 各区域之间有充足的空间
- **padding**: 通常使用较大的padding (80-120px垂直)
- **元素间距**: 适中，不拥挤

### 5. 3D地球实现
- **WebGL/Three.js**: 使用3D渲染
- **特点**:
  - 深蓝色地球
  - 发光的网格线
  - 数据点/节点动画
  - 缓慢旋转
  - 光晕效果

### 6. 颜色方案
- **背景**: 深蓝黑色 (#0a0a1a 或类似)
- **主色**: 蓝色 (#0090ff)
- **强调色**: 青色 (#00d4ff)
- **文字**: 白色/浅灰色

## 应用到LatentMind

### 需要调整的内容
1. 减小标题字体大小
2. 添加固定透明导航栏
3. 添加3D地球动画（使用Three.js或CSS动画模拟）
4. 增加留白空间
5. 调整整体比例更舒适
